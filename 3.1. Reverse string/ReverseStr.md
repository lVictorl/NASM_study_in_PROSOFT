# Разворот строки 
## Назначение и описание
Программа читает строку с клавиатуры (максимум 100 символов) и выводит её в обратном порядке.

Данный проект представляет собой полнофункциональную программу на языке ассемблера, которая демонстрирует работу со строками и взаимодействие с пользователем через консоль. Программа запрашивает у пользователя ввод строки, разворачивает её задом наперед и выводит результат. Особенностью проекта является комплексная обработка ошибок и модульная структура с выделенными функциями для различных операций.

Основной функционал включает интерактивный диалог с пользователем: вывод приглашения к вводу, чтение строки с клавиатуры, валидацию входных данных (проверка на пустую строку и превышение максимальной длины), разворот строки и вывод результата. Программа поддерживает строки длиной до 100 символов и корректно обрабатывает различные ошибочные сценарии, включая проблемы с чтением/записью и некорректный пользовательский ввод. Для работы со строками реализованы специализированные функции: удаление символа новой строки, разворот строки, проверка на пустоту и вычисление длины.

Архитектурные соображения показывают хорошо продуманную структуру программы с четким разделением на основные модули: главную программу, обработчики ошибок и вспомогательные функции. Использование сегментов .data для констант и .bss для буферов демонстрирует правильное управление памятью. Однако стоит отметить, что программа использует 32-битные системные вызовы через int 0x80, что делает её несовместимой с 64-битными системами без адаптации. Для промышленного использования рекомендуется добавить более детальную диагностику ошибок и возможность обработки Unicode-символов, но в качестве учебного примера программа отлично демонстрирует принципы работы со строками в ассемблере.


# Краткое описание структуры программы
### Основные функциональные блоки:
_start - главная точка входа программы

remove_newline - удаляет символ новой строки из входной строки

reverse_string - разворачивает строку задом наперёд

check_empty_string - проверяет, является ли строка пустой

get_string_length - вычисляет длину строки


### Обработчики ошибок:
read_error - ошибка чтения из stdin

write_error - ошибка записи в stdout

empty_error - введена пустая строка

too_long_error - строка превышает 100 символов
